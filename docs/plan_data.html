Final Submission Format

The Plan Data Sheet compiles bearings, distances, and beacon labels into one table formatted specifically for submission to the Lands Commission. It removes the need to copy from multiple sheets.

Key Columns

  • From Point & To Point
  • Bearing (DMS)
  • Distance (meters)
  • Remarks (Area, Notes)

Each entry is structured to match regional plan submission layouts in Ghana, ensuring faster approvals and fewer corrections.

Multi-Use Benefits

Beyond approval, this format is ideal for internal verification, engineering layout preparation, and contract documentation. It can be printed, signed, or exported as a PDF with no extra formatting required.
