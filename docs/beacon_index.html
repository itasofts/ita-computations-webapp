What is a Beacon Index?

The Beacon Index is a critical part of any fieldwork documentation. It lists all coordinate points (beacons) with their corresponding X (Eastings) and Y (Northings), alongside fields like sheet number, record book, and remarks. This sheet acts as the master reference for all other computations.

Traditional vs Digital

Surveyors previously prepared this sheet manually, entering each beacon one by one from field notes. Errors in digit order or missed entries often delayed submissions. With ITA Computations, this process is fully automated. Once coordinates are entered, the Beacon Index is generated instantly and formatted for submission.

Editable Fields for Compliance

Users can insert metadata like:

  • Computation Sheet Number
  • Record Book Number
  • Record Book Page
  • Remarks or Notes
