Welcome

ITA Computations is a lightweight, offline-first geospatial app built to meet the computation needs of Ghanaian land surveyors. Whether you're on-site or preparing documents for approval, this tool supports bearing & distance, area computation, plan data generation, and more.

How It Works

The web version includes modular computation pages where you can:

  • Enter coordinates manually, paste, or import CSV
  • Calculate bearing & distance with auto-generated sheets
  • Generate area computation and plan data summaries
  • Print or export as PDF for submission or record keeping
